# Crop Yield Prediction

A modern Streamlit regression dashboard for crop yield prediction. The entire application lives in a single file, `pipeline.py`.

## Run

```bash
pip install -r requirements.txt
streamlit run pipeline.py

```

## Features

- CSV upload or built-in sample crop dataset
- Data analysis with summary statistics, missing values, and correlation heatmap
- Data preprocessing with automatic cleaning and feature selection
- Visualization for rainfall, temperature, crop-wise yield, area vs yield, and actual vs predicted values
- Regression model training with Linear Regression and Random Forest Regressor
- Model evaluation using R2 Score, MAE, MSE, and 5-fold cross-validation
- Prediction form for crop yield estimation
- SQLite history for saved predictions
